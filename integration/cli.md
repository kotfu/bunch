# The Bunch CLI

The latest version is {% bunchcli version %}.

Install the Bunch CLI using `gem install bunchcli`. Run `bunch -h` for usage instructions.

{% bunchcli help %}

Example:
    
    # Open or Toggle Coding.bunch
    bunch Coding

    # Force open Coding.bunch
    bunch -o Coding
