# Dark Mode Commands

Turn macOS Dark Mode on and off: `(dark mode on)`, `(dark mode off)`. An "on" command can be shortened to just `(dark mode)`, and `(dark mode off)` can be shortened to `(light mode)`.

## Examples

```
(dark mode)
(dark mode off)
```
