## Dock Commands
{:.text-alpha}

`(hide dock)` and `(show dock)` hide and show the Dock (toggle Hiding). Reposition the dock with `(dock left)`, `(dock bottom)`, or `(dock right)`.

### Examples
{:.text-beta}

```bunch
(show dock)
(dock bottom)
```

## Desktop Commands
{:.text-alpha}

Hide and show desktop icons: `(hide desktop)`, `(show desktop)`.

### Examples
{:.text-beta}

```bunch
(show desktop)
(hide desktop)
```
