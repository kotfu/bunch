## Dock Commands

(`(hide dock)`, `(show dock)`) show and hide the Dock (Turn Hiding On), or reposition the dock (`(dock left)`, `(dock bottom)`, `(dock right)`).

## Examples

```
(show dock)
(dock botttom)
```

## Desktop Commands

Hide and show desktop icons: `(hide desktop)`, `(show desktop)`.

## Examples

```
(show desktop)
(hide desktop)
```
